// 03 — Projects

Things I've built

01 /

Housing Migration Analytics Platform

Designed and implemented a normalized SQLite analytics database containing 395K+ U.S. housing, migration, AHS, and CHAS records. Built a schema-aware RAG pipeline using OpenAI embeddings to generate safe, explainable SQL — preventing hallucinations and enabling non-technical users to safely explore complex housing data without writing SQL.

SQL RAG Pipeline OpenAI Embeddings SQLite Python
↗

02 /

Predictive Pavement Condition Analysis

Processed and analyzed 2,000+ pavement condition records using imputation, normalization, and outlier handling. Applied K-Means clustering to identify three deterioration patterns including faster degradation in high-traffic environments. Best performance achieved with Gradient Boosting (RMSE ≈ 890), enabling proactive infrastructure maintenance planning.

Gradient Boosting K-Means Scikit-learn Big Data Python
↗

03 /

Obesity Level Analysis

Conducted EDA on 2,000+ records to uncover relationships between BMI, lifestyle factors, and family history. Developed and validated Logistic Regression, Decision Tree, SVM, and Clustering models — achieving up to 85% accuracy with SVM. Findings translated into actionable public-health insights around diet, physical activity, and family history.
